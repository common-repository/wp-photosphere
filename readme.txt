Bindet via Shortcode eine Photosphere-Aufnahme über die Google+-API in Artikeln und Seiten ein.


== Description ==

"WP-Photosphere" ermöglicht es, via Shortcode und der Google+-API eine Photosphere-Aufnahme in Artikeln und Seiten einzubinden.

= Einbindung =

Die Einbindung erfolgt via Shortcode mittels Button im WYSIWYG-Editor, dabei wird lediglich die URL des Bildes benötigt. Der Shortcode an sich lautet:

[photosphere]URL-des-Bildes[/photosphere]

Dazu gibt es noch ein paar Optionen, welche sich in den Plugin-Einstellungen auch dauerhaft bestimmen lassen. Diese lauten width=600, height=400 und autorotate (jeder Wert ist optional), kleines Beispiel mit allen drei Werten:

[photosphere width=600 height=400 autorotate]URL-des-Bildes[/photosphere]
